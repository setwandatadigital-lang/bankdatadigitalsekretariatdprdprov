PUSAT PANDUAN INTEGRASI & DEPLOYMENT

Sistem ini siap di-deploy secara instan tanpa biaya server bulanan.

Panduan Penggunaan Operasional (User Manual)

1 Manajemen Kategori Arsip Baru (18 Kategori Terintegrasi)

Sistem kini mengelola 18 kategori arsip baru yang spesifik. Saat Anda mengunggah dokumen baru atau melakukan scan OCR, pastikan memilih salah satu kategori yang presisi seperti Dokumen SPJ Kontraktual, Dokumen Perjadin Keg Reses, atau Dokumen Bagian TU & Keuangan. Dokumen akan langsung diklasifikasikan dengan sistem pengkodean visual warna dinamis.

2 E-Disposisi Elektronik Real-Time

Gunakan akun dengan peran Sekretaris DPRD (Sekwan) atau Super Admin untuk membuka surat masuk di tab "Surat Menyurat", lalu klik tombol E-Disposisi. Anda dapat meneruskannya ke pejabat tujuan (misal: Kabag Umum) disertai dengan petunjuk instruksi khusus. Alur surat secara otomatis terupdate statusnya menjadi "Sudah Disposisi".

3 Penerapan Tanda Tangan Elektronik (TTE / E-Sign)

Klik tombol E-Sign pada baris surat manapun. Buat goresan tanda tangan Anda langsung di dalam kotak canvas yang disediakan. Canvas ini telah dioptimalkan secara responsif untuk layar ponsel cerdas dan tablet dewan pimpinan, sehingga mempermudah persetujuan dinas jarak jauh. Klik "Terapkan Tanda Tangan" untuk mengunci data secara kriptografis pada sistem lokal.

4 Simulasi OCR Scanner Cerdas

Pada tab "Dokumen Digital", klik tombol OCR Scan Simulator. Anda dapat memilih dokumen fisik (gambar atau dokumen pdf hasil scan) lalu sistem akan memicu simulasi pemindaian AI untuk mengekstrak isi teks, nomor surat, dan perihal secara instan yang dapat langsung diimpor ke dalam basis data digital.

Panduan Deployment Mandiri

Metode 1: Google Apps Script (Rekomendasi Utama)

Metode ini gratis selamanya, tanpa server VPS, serta memanfaatkan Google Drive dan Google Sheets milik instansi Anda sebagai database aman.

  1. Buat file **Google Sheets** baru di Google Drive instansi Anda.
  2. Klik **Ekstensi** > **Apps Script**.
  3. Tulis kode skrip kontroler `Code.gs` seperti di bawah ini.
  4. Buat file HTML baru dengan nama `index.html`, lalu salin seluruh kode dari aplikasi tunggal ini.
  5. Klik **Terapkan (Deploy)** > **Penerapan Baru** > Pilih **Aplikasi Web** > Atur Akses ke **Siapa Saja (Anyone)**.
function doGet() {
  return HtmlService.createHtmlOutputFromFile('index')
    .setTitle('BANK DATA DIGITAL SEKRETARIAT DPRD')
    .setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL);
}
Metode 2: Netlify / Vercel / GitHub Pages

Karena aplikasi ini merupakan *pure single-page client application*, Anda cukup mengunggah satu file ini (`index.html`) langsung ke Netlify Drop atau GitHub repository. Layanan hosting statis gratis akan langsung aktif dalam hitungan detik dengan dukungan protokol keamanan SSL/HTTPS otomatis.

└── project-root/
    └── index.html (File tunggal hasil unduh/copy dari sistem ini)
Metode 3: Integrasi Blogger (Post/Pages)

Jika instansi memiliki website beralaskan Blogger, Anda dapat memasang aplikasi web ini di dalam halaman statis.

  1. Buat Halaman Baru (Page) di dashboard Blogger.
  2. Ubah mode ketik ke **Tampilan HTML (HTML View)**.
  3. Paste seluruh baris kode ini lalu publikasikan.
